# SQL Core Banking Basics

SQL fundamentals practice using a simplified core banking data model.

## Tables
- customers
- accounts
- transactions

## Files
- schema.sql: table definitions
- data.sql: sample data
- queries.sql: practice queries (filters, joins, aggregations, unions, string functions)

## How to run
1. Run `schema.sql`
2. Run `data.sql`
3. Run `queries.sql`
